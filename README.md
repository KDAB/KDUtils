# KDUtils

A set of C++ helpers and wrappers around the C++ standard library

You get:

* ByteArray
* Dir
* ElapsedTimer
* File
* Flags
* Url
* Logger

## Building

On Linux:

```bash
sudo apt install libxkbcommon-dev libxcb-xkb-dev libxkbcommon-x11-dev wayland-scanner++ wayland-protocols
```
